# Termcase [v1.0]
Termcase is a command line interface cs2 case simulator.

# Installation

[🐧] Linux: make sure you have git and python installed before. Type "git clone https://github.com/pattbyte/Termcase", cd to Termcase and type execute the termcase.py file using python.

[🪟] Windows: clone the repository (with the same command) cd to it, and execute termcase.py using python.

[📱] Android: same as with linux, make sure you have git and python, and follow the same steps. MUST have termux installed.
